RapidCars — Performance Car Rental Website
Project Overzicht

RapidCars is een moderne autoverhuur website ontwikkeld voor een klant die zich richt op het verhuren van sportieve auto's aan een jonge doelgroep. Het doel van de website is om bezoekers snel het aanbod van auto's te laten bekijken en eenvoudig beschikbaarheid aan te vragen via directe contactopties zoals WhatsApp of een contactformulier.

De website combineert een minimalistisch design met een duidelijke gebruikersflow zodat bezoekers snel van auto bekijken → beschikbaarheid aanvragen → contact opnemen kunnen gaan.

Doel van het project

De klant had nog geen professionele online aanwezigheid en wilde een moderne website waarmee potentiële klanten:

het aanbod van auto's kunnen bekijken

details van auto's kunnen zien

snel beschikbaarheid kunnen aanvragen

eenvoudig contact kunnen opnemen

De focus lag op snelheid, duidelijkheid en mobiel gebruik, omdat een groot deel van de doelgroep de website via hun smartphone bezoekt.

Oplossing

Voor RapidCars heb ik een moderne single-page webapplicatie ontwikkeld met React, TypeScript en Vite. De website bestaat uit meerdere pagina’s en een duidelijke navigatiestructuur zodat bezoekers eenvoudig door het aanbod kunnen browsen.

De gebruikersflow van de website is ontworpen als:

Homepage → Aanbod → Auto detailpagina → Beschikbaarheid aanvragen → Contact

Deze structuur zorgt ervoor dat bezoekers snel informatie vinden en direct actie kunnen ondernemen.
